# 童心教育機構-後台管理系統👋
 
 這是一個使用虛擬的後台管理系統*童心教育機構*。

 使用者可以在後台完成訂單管理流程與銷貨分析與圖表展示。

 ## 功能介紹👋

- 後台

  - 首頁：

    - 列出學生人數、開班數與營業額相關訊息
    - 圖表顯示近半年的班級狀況

  - 訂單列表：

    - 分門別類，分為全部訂單、已處理、未處理與退款選項
    - 以列表的方式呈現所有訂單狀況，並可修改訂單或刪除訂單

  - 單一訂單細節：

    - 分為摘要、課程清單、發票管理與危險操作等細項
